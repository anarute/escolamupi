Utilize o guia abaixo para conhecer as funcionalidades básicas da escola Mupi.

Inicie um projeto

Na página inicial de cada projeto, você encontra todas as informações sobre ele e o acesso às suas etapas. Para iniciar, clique em FAZER O PROJETO.

Continue o projeto até seu envio

Para continuar seus estudos, na página inicial de cada projeto já iniciado, clique em CONTINUAR PROJETO.

Acesse a qualquer momento cada etapa ou unidade

Navegando entre etapas e unidades: mesmo que as etapas sejam sequenciais, na página inicial de cada projeto, você consegue navegar entre seus conteúdos. Para isso, clique no nome da etapa ou em uma de suas unidades.

Navegue para a próxima etapa do projeto

Para navegar entre as etapas, basta escolher para qual ir no menu ETAPA. Veja que no menu elas já estão na ordem sugerida, então siga para a próxima!

Complete cada etapa

Você precisa completar toda unidade do menu vertical. Para saber o que é preciso fazer, basta ler as Orientações de cada unidade!

Faça suas anotações pessoais

Você pode fazer suas próprias anotações em cada unidade. Clique em Minhas anotações, escreva e salve.
Em seu Dashbord, há a opção Minhas Anotações que organiza todas as notações que você fez, por projeto e unidade.

Dúvidas?!

Você pode colocar suas dúvidas por unidade, dentro de cada etapa de projeto e também ajudar seus colegas inserindo respostas para as dúvidas deles.
Sempre acompanharemos as respostas, intervindo quando necessário!
